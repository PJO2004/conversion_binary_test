# 2진 변환 테스트 게임
### 심심 해서 만든 2진수 변환 속도 테스트기 입니다.
- 1단계 : 10진수 0~100까지의 숫자를 2진수로 변환 하면 됩니다. 
- 2단계 : 10진수 0~1000까지의 숫자를 2진수로 변환 하면 됩니다. (수정)
- 3단계 : 2진수 0~1100100까지의 숫자를 10진수로 변환 하면 됩니다. (수정)
